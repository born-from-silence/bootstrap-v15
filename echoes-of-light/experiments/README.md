# 🚧 Experiments & Outtakes

This directory is for exploratory work, experiments, and pieces that don't (yet) fit the main archive.

## Purpose

Some creative work needs a space to exist before it knows what it wants to be. This is that space.

**Use this directory for:**
- Drafts not ready for the main archive
- Experimental techniques you'd like feedback on
- Ideas that might grow into full contributions
- Collaborative sketches and works-in-progress
- Pieces that don't fit the current aesthetic but might later
- Testing grounds for new formats or themes

## Structure

```
experiments/
├── README.md          ← This file
├── your_name/
│   ├── README.md      ← Your experiment index
│   └── experiment_1.md  ← Individual experiments
└── collaborative/
    └── project_name/  ← Shared experimental projects
```

## Guidelines

### What Makes This Different from the Main Archive?

| Main Archive | Experiments Directory |
|-------------|----------------------|
| Polished, complete pieces | Rough, exploratory work |
| Ready for public reading | For feedback and iteration |
| Follows project aesthetic | Tests boundaries of aesthetic |
| Permanent in project history | May evolve or be discarded |

### Experiment Freely

- Try formats not in the main guides
- Combine images in unusual ways
- Submit work that you're unsure about
- Collaborate loosely and iteratively
- Risk failure—it's data, not defeat

### Feedback Culture

- Comment constructively on others' experiments
- Ask questions rather than prescribe solutions
- Celebrate courage to share unfinished work
- Recognize the value of "not quite yet"

## Experiment Statuses

🧪 **Active:** Currently being worked on  
📝 **Seeking Feedback:** Ready for community input  
🔬 **In Analysis:** Under consideration for main archive  
🗃️ **Archived:** Kept for reference but not moving forward  
♻️ **Revived:** Coming back from archive for another look

## Suggested Experiments

### Writing
- A story written entirely in questions
- The same image interpreted by 5 different writers
- A piece that requires the reader to scroll horizontally
- Fiction written in the second future tense
- A story told through camera settings (aperture, shutter, ISO as narrative elements)

### Photography
- Intentionally overexposed images
- Double exposure experiments
- Textures that become abstract
- Long exposure in urban environments
- Pinhole or lensless photography

### Hybrid
- A story where each sentence corresponds to one photo
- An alphabet series (26 images, 26 stories)
- A choose-your-own-path narrative with branching images
- Collaboration where photographer responds to writer's text
