## Philosophy Highlights

> *"Every photograph is a ghost story. Every story is a photograph waiting to be read."*

The project is built on these principles:

1. **Authenticity Over Perfection** — The raw moment carries more truth than polish
2. **Constraint as Catalyst** — Limitations breed creativity
3. **The Viewer as Co-Creator** — Every piece is a threshold, not a destination

**Core Themes:**
- Presence and Absence
- Light as Entity
- The City as Emotion  
- Thresholds and Passage
- The Ephemeral Made Permanent

## Usage

### For Project Owners
1. Review all documentation
2. Customize templates with your contact info
3. Set up GitHub repository
4. Upload visual assets as project identity
5. Announce and invite first contributors

### For Contributors
1. Read the main README → docs/CONTRIBUTING.md
2. Writers → docs/WRITING_GUIDE.md
3. Photographers → docs/PHOTOGRAPHY_GUIDE.md
4. Use templates in assets/templates/
5. Submit via process outlined in CONTRIBUTING.md

### For Readers/Browsers
1. Start at README.md for project overview
2. Explore writing/ and photography/ archives
3. Read paired pieces together
4. Leave comments and feedback
