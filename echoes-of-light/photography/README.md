# 📷 Photography Archive

This directory contains the photographic prompts waiting for written interpretation.

## Structure

```
photography/
├── README.md           ← This file
├── categorized/        ← Organized by theme/type
│   ├── urban/
│   ├── nature/
│   ├── domestic/
│   ├── abstract/
│   └── portraits/
├── submitted/          ← New submissions awaiting cataloging
├── sample/            ← EXAMPLE: Blue Hour on Mercer Street
│   ├── blue_hour_mercer_street.md
│   └── blue_hour_mercer_street.svg
└── metadata.json       ← Machine-readable catalog
```

## Image Status

🟢 **Open:** Available for written response  
🟡 **Claimed:** Writer has reserved this image (in progress)  
🔵 **Paired:** Has at least one written echo  
🟣 **Featured:** Highlighted in collections or special showcases

---

## Technical Requirements

- **Format:** JPEG, high quality (80-95%)
- **Resolution:** Minimum 2048px longest edge
- **Color:** Color or B&W as suits the image
- **Metadata:** Title, date, location, camera, context

## Submission Template

```markdown
# Title of Your Image

**Filename:** your-image.jpg  
**Status:** 🟢 Open  
**Date Taken:** YYYY-MM-DD  
**Location:** City, Country (or vague)  
**Camera/Equipment:** [Device/lens/settings if notable]  
**Themes:** [comma-separated from Story Bible]

---

**What Drew Me:**  
2-3 sentences about the moment, the light, the feeling.

**What I See:**  
The potential narrative threads—questions, characters, possibilities.

**What I Left Out:**  
The intentional absences—the mystery waiting for words.

**Technical Context:**  
Any technical notes about light, composition, post-processing.

---

**Photographer:** Your Name  
**Contact:** [optional]  
**Portfolio:** [link if desired]
```

## For Writers

1. Browse images by category or scroll the gallery
2. Check status (🟢 open for writing)
3. Read metadata—the photographer's intentions matter
4. Claim by commenting or opening an issue
5. Write your echo in `/writing`

---

## For Photographers

1. Review PHOTOGRAPHY_GUIDE.md
2. Select your strongest narrative images
3. Prepare metadata thoughtfully
4. Submit via file upload or cloud link
5. Engage with writers who respond
