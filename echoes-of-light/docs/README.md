## Quick Reference: Story Bible Themes

1. **Presence and Absence** — Ghosts, traces, what remains
2. **Light as Entity** — Illumination as actor, not backdrop
3. **The City as Emotion** — Urban spaces as inner landscapes
4. **Thresholds and Passage** — Doorways, transitions, becoming
5. **The Ephemeral Made Permanent** — Freezing time's flow

## Workflow Summary

### For Writers
```
1. Browse → /photography for available images (🟢 Open)
2. Select → Image that sparks a story
3. Write → Use WRITING_TEMPLATE.md
4. Submit → Via PR or email to /writing
```

### For Photographers
```
1. Shoot → Following PHOTOGRAPHY_GUIDE.md
2. Select → 5-10 best narrative images
3. Document → Use PHOTOGRAPHY_TEMPLATE.md
4. Submit → High-res files + metadata
```

### For Collaborators
```
1. Connect → Find partner via Discussions/Issues
2. Align → Share vision and timeline
3. Create → Iterative back-and-forth
4. Submit → Joint credit, mutual satisfaction
```
