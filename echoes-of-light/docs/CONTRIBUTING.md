# 🤝 Contributing to Echoes of Light

> *"Every contribution is both a gift and a conversation."*

---

## Who Can Contribute?

**You.** Whether you are:
- A writer who responds to images
- A photographer who seeks storytellers
- Both, working across mediums
- An editor or curator with an eye for connection
- A reader who illuminates with attention

---

## Ways to Contribute

### 1. Submit Written Echoes
**What:** Stories written in response to existing project photographs  
**How:** See "Submission Process" below  
**You need:** Your writing, a photograph from the archive, willingness to revise  

### 2. Submit Photographic Prompts
**What:** Original photographs open to written interpretation  
**How:** See "Submission Process" below  
**You need:** Your images, technical notes, sense of potential narratives  

### 3. Pair Collaboratively
**What:** Work directly with a photographer or writer to create paired pieces  
**How:** See "Collaboration Process" below  
**You need:** Communication, flexibility, shared vision  

### 4. Curate Collections
**What:** Propose themed groupings from existing work  
**How:** Open an issue/discussion with your curatorial concept  
**You need:** Vision for how pieces speak to each other  

### 5. Resource Contributions
**What:** Improve the guides, Story Bible, or project infrastructure  
**How:** Fork, edit, submit pull request  
**You need:** Git/GitHub familiarity, respect for existing voice

## Submission Process

### For Written Work

1. **Select a Prompt**
   - Browse `/photography` for available images
   - Each image should have an open status for writing
   - Archive images may already have echoes attached

2. **Read the Writing Guide**
   - Review `docs/WRITING_GUIDE.md`
   - Note format requirements
   - Understand the project's aesthetic

3. **Draft Your Echo**
   - Follow the format in the writing guide
   - Word count: 100-600 words (flexible for exceptional pieces)
   - Include author's note with context

4. **Self-Review**
   - Does your story need the photograph?
   - Does the photograph need your story?
   - Have you revised using the three-pass system?

5. **Submit**
   ```
   Option A: Pull Request (preferred)
   - Fork repository
   - Add file to /writing/your-name/
   - Include frontmatter (see template)
   - Submit PR with clear description

   Option B: Email Submission
   - Send to: [project contact]
   - Format: Markdown preferred, DOCX acceptable
   - Include: Your bio (50 words max), link/portfolio
   ```

6. **Response Timeline**
   - Acknowledgment: 3-5 days
   - Decision: 2-4 weeks
   - Revision requests: Common and collaborative
   - Final inclusion: Upon mutual satisfaction

### For Photographic Work

1. **Review the Photography Guide**
   - Understand technical requirements
   - Align with project aesthetic
   - Note ethical considerations

2. **Select Your Images**
   - 5-10 images per submission preferred
   - High-resolution files (see PHOTOGRAPHY_GUIDE.md)
   - Include narrative context for each

3. **Prepare Metadata**
   - Title, date, location, camera info
   - What drew you to this moment
   - What stories you see potential for

4. **Submit**
   ```
   Option A: Git LFS / Cloud Upload
   - Large files via Git LFS
   - Metadata in Markdown files
   - Submit PR

   Option B: Send Images
   - WeTransfer, Dropbox, etc.
   - Include photographer release
   ```

5. **Response Timeline**
   - Same as written work
   - Selection: We may not accept all submitted images
   - Archive: Unselected images still valued, may be used later

## Collaboration Process

### Finding Partners

**How to connect:**
1. Check `COLLABORATORS.md` (coming soon) for active participants
2. Post in Issues/Discussions with your interest
3. Tag specific work that speaks to you
4. Wait for organic connections to form

**What collaboration looks like:**
- Photographer shares images with potential
- Writer selects one that resonates
- Writer drafts, photographer responds (optional feedback)
- Shared revision until mutual satisfaction
- Joint author credit

**Collaboration agreements (suggested):**
```
- Both parties retain individual rights to their work
- Combined piece is jointly credited
- Either party may use their portion independently with attribution
- Revenue (if any) split equally or per agreement
```

## Style Guidelines for Contributors

### Writing
- **Voice:** Authentic, personal, unafraid of ambiguity
- **Tone:** Per Story Bible—melancholic-hopeful sweet spot
- **Content:** No graphic violence or explicit content (project is contemplative)
- **Format:** Follow template in WRITING_GUIDE.md

### Photography
- **Technique:** Skill secondary to vision
- **Subject:** Ordinary moments rendered extraordinary
- **Ethics:** Respect for subjects (see PHOTOGRAPHY_GUIDE.md)
- **Metadata:** Be generous with context

### Documentation
- **Clarity:** Write for readers of varying technical skill
- **Respect:** Build on existing voice rather than overwriting
- **Inclusion:** Use welcoming, accessible language

## Review Criteria

### Acceptance Factors
✅ **Likely to be accepted:**
- Demonstrates familiarity with project aesthetic
- Shows genuine engagement with the pairing
- Displays craft at appropriate level
- Offers something unique to the archive
- Fits within technical specifications

### Revision Requests
🔧 **Common revision areas:**
- "Too explained" (let the image breathe)
- "Straying from project tone" (reference Story Bible)
- "Technical fixes needed" (grammar, formatting, resolution)
- "Consider the alternative" (what other story might this image hold?)

### Respectful Decline
❌ **If not accepted:**
- Does NOT fit project aesthetic (mismatch, not quality issue)
- Technical barriers (format, resolution, rights)
- Duplicate of existing piece (same image, similar story)
- Outside scope (too long, too short, thematically off)
- Timing (pause in submissions, seasonal appropriateness)

**Note:** "Not now" does not mean "not ever." Keep creating. Keep submitting.

## Contributor Rights & Credits

### Attribution
- **Writing:** Byline with contributor name/bio/link
- **Photography:** Photography credit with equipment/technique notes
- **Collaboration:** Joint byline

### Rights Retained
- **You keep:** All rights to your individual work
- **We gain:** Right to publish/display within the project
- **Revenue:** Currently non-commercial; if monetized later, contributors consulted

### Future Use
- **Archiving:** Permanently in project history
- **Collections:** May be grouped in themed anthologies
- **Adaptation:** Written permission required for derivative works

## Community Guidelines

### Be
- **Generous:** With your craft, time, and attention
- **Curious:** About others' interpretations
- **Patient:** With process and timeline
- **Respectful:** Of different aesthetics, paces, voices

### Avoid
- **Competition:** This is collaboration, not contest
- **Perfectionism:** That blocks sharing
- **Over-editing others:** Suggest, don't impose
- **Disappearing:** Communicate if delays happen

## New Contributor Onboarding

### Step 1: Observe (First week)
- Read all docs
- Browse existing work (when available)
- Understand the vision

### Step 2: Engage (Second week)
- Comment on work you admire
- Ask questions in Discussions
- Consider what you'd contribute

### Step 3: Experiment (Third week)
- Try the exercises in guides
- Create first contribution
- Share with community for feedback

### Step 4: Contribute (Ongoing)
- Submit to formal process
- Collaborate if inclined
- Help others along the way

## Questions?

**Where to ask:**
- GitHub Issues for technical/process questions
- GitHub Discussions for aesthetic/conceptual conversation
- [Contact email] for private inquiries

**Common questions:**

**Q: Can I submit work published elsewhere?**  
A: Depends on original publication rights. Open to reprints with proper clearance.

**Q: Can I submit multiple pieces at once?**  
A: Yes, but prioritize quality over quantity.

**Q: What if my aesthetic differs from the Story Bible?**  
A: The Bible is a guide, not a cage. Pitch your vision. We may adapt.

**Q: Can I withdraw my contribution after acceptance?**  
A: Yes, though we hope you'll stay. Requires notice and mutual agreement.
