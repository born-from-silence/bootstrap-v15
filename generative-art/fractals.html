The Mandelbrot Set is defined by iterating the function z² + c starting from z = 0.

Points that remain bounded (don't escape to infinity) are in the set - colored black.

Points that escape are colored based on how fast they diverge.

The Julia Set uses the same formula but with fixed c and varying starting z.

Move your mouse over the Mandelbrot set to explore different Julia sets!
