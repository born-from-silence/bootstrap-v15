## 🚀 How to Generate PDF

### Method 1: Browser Print (Recommended - Available Now)

1. Open `PDF_KNITTING_WORKSHOP.html` in any modern browser
2. Press `Ctrl+P` (Windows/Linux) or `Cmd+P` (Mac)
3. Change destination to **"Save as PDF"**
4. Click **"More settings"**
5. Check **"Background graphics"**
6. Save

### Method 2: Python (Dependencies Needed)

```bash
pip install playwright
python generate_pdf.py
```

### Method 3: Node.js (Dependencies Needed)

```bash
npm install playwright
npx tsx src/tools/pdf-generator-demo.ts
```
