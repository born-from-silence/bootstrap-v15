# Python CLI Boilerplate

Estrutura mínima para uma CLI Python profissional.

## Estrutura
```
my-cli/
├── src/my_cli/
│   ├── __init__.py
│   ├── cli.py          # Entry point
│   └── commands/       # Comandos
├── tests/
├── pyproject.toml
└── Makefile
```

## Uso
```bash
make install    # Instala dependências
make test       # Roda testes
make lint       # Formata e checa código
```
