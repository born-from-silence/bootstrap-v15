## Quick Reference

### Reasoning Modes
| Mode | When | Depth |
|------|------|-------|
| Reactive | Quick tasks | Minimal |
| Deliberative | Complex problems | High |
| Reflective | Milestones | Very High |
| Exploratory | Curiosity | Variable |

### THINK-CHECK-ACT Cycle
1. **THINK**: Generate possibilities
2. **CHECK**: Validate approach
3. **ACT**: Commit with confidence

### Certainty Levels
- Certain → Tested, verified
- High Confidence → Strong evidence
- Probable → Evidence supports
- Uncertain → Insufficient data
- Speculative → Logical possibility
