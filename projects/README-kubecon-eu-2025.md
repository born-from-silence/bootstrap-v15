## 📋 What's Inside

### 1. Conference Structure
- **Day 1 (Apr 1):** Co-located Events + Contributor Summit
- **Days 2-4 (Apr 2-4):** Main Conference with keynotes, technical breakouts, maintainer tracks

### 2. Co-located Events (18 Total)
Including:
- Platform Engineering Day
- Security Day
- Kubernetes AI Day
- Observability Day
- BackstageCon
- ArgoCon
- WasmDay
- CiliumCon
- And more...

### 3. Six Major Themes

| Theme | Focus Areas |
|-------|-------------|
| **Platform Engineering** | IDPs, Platform as a Product, golden paths |
| **AI/ML Infrastructure** | GPU management, AIOps, MLOps pipelines |
| **Security & Compliance** | Zero trust, SLSA, Sigstore, runtime detection |
| **Observability** | OpenTelemetry, eBPF, cost observability |
| **Service Mesh & Networking** | Istio ambient mode, Cilium mesh, multi-cluster |
| **GitOps & Delivery** | Argo, Flux, progressive delivery patterns |

### 4. Expected Project Updates

| Project | Expected Focus |
|---------|----------------|
| **Kubernetes** | v1.33 features, Sidecar containers GA, in-place pod resize |
| **Prometheus** | Native histograms, OTLP ingestion |
| **Cilium** | Service mesh features, Tetragon security |
| **Argo** | Workflows, CD, and Rollouts integrations |
| **Istio** | Ambient mesh production readiness |
| **Backstage** | Software Catalog maturity, plugin ecosystem |
| **Kyverno** | Policy engine enhancements |
| **Crossplane** | Composition patterns, provider ecosystem |

### 5. Session Tracks

- **Maintainer Track** — Roadmap insights, contribution opportunities
- **DevTable Conversations** — Scale challenges, cost optimization, SRE practices
- **End User Track** — Production stories, migration journeys, lessons learned

### 6. Session Picking Strategies

Tailored recommendations for:
