# Building a Voice-Enabled AI Agent

Great question! Let me break down the architecture and options for building a voice-enabled AI agent.

## Core Architecture

A voice-enabled AI agent requires **three main components**:

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   Speech-to-    │     │   AI Agent/LLM   │     │   Text-to-      │
│   Text (STT)    │────▶│   (Processing)   │────▶│   Speech (TTS)  │
└─────────────────┘     └──────────────────┘     └─────────────────┘
   *Listen/Transcribe*    *Think/Respond*           *Speak*
```

---

## 1. Speech-to-Text (STT) Options

### **Cloud APIs** (Easiest)
| Service | Pricing | Features |
|---------|---------|----------|
| **OpenAI Whisper** | $0.006/minute | Best accuracy, supports 99 languages |
| **Google Cloud Speech** | $0.024/minute (first 60 min free/mo) | Real-time streaming, good for production |
| **Azure Speech** | $1/hour audio | Enterprise features, custom models |
| **AWS Transcribe** | $0.024/minute | AWS integration, medical/legal variants |

### **Self-Hosted** (Privacy/Control)
- **Whisper.cpp** - OpenAI's model optimized for C++
- **Faster-Whisper** - Python, faster than original
- **Vosk** - Offline, lightweight

### **Real-time Streaming**
For live conversation, you need **streaming STT**:
- WebRTC audio stream → chunked processing
- Interim results for responsive feel
- Endpointing to detect silence

---

## 2. AI/LLM Processing

Once you have text, you process it like any chatbot:
- **OpenAI GPT-4** - Standard, good reasoning
- **Claude** - Better for longer contexts, nuance
- **Local models** (Llama, Mixtral) - Privacy, cost control

**Latency optimization:**
- Stream tokens as they're generated
- Use shorter context windows for speed
- Cache common responses

---

## 3. Text-to-Speech (TTS) Options

### **Neural/High Quality**
| Service | Voice Quality | Cost |
|---------|--------------|------|
| **Eleven Labs** | **Best** - ultra realistic | $5/month for 30 minutes |
| **OpenAI TTS** | Very good | $0.015 per 1K characters |
| **Azure Neural** | Good, enterprise | ~$16 per 1M characters |
| **Amazon Polly** | Good, affordable | ~$16 per 1M chars (neural) |

### **Open Source**
- **Coqui TTS** - Python, customizable
- **Piper** - Fast, runs locally
- **Mimic 3** - Mycroft's TTS

## 4. Implementation Approaches

### **A. Async/Sequential** (Simpler)
```javascript
// Pseudocode
while (conversation_active) {
  audio = record_audio();          // Wait for speech
  text = await stt.transcribe(audio);
  response = await llm.chat(text);
  audio_out = await tts.synthesize(response);
  play(audio_out);                  // Wait for playback
}
```
**Latency:** 2-5 seconds per turn
**Best for:** Commands, Q&A, non-realtime use

### **B. Streaming/Realtime** (Complex, better UX)
```
User: [speaking continuously]
  ↓ (stream chunks)
STT: partial_1 → partial_2 → [END OF UTTERANCE]
  ↓ (send to LLM immediately once sentence complete)
LLM: generating → generating...
  ↓ (stream tokens to TTS as they arrive)
TTS: [audio chunk 1] → [audio chunk 2]...
  ↓
Speaker: plays with minimal latency
```
**Latency:** <1 second first-audio, continuous flow
**Best for:** Conversational AI, voice assistants

## 6. Practical Code Examples

### **Python: Full Voice Agent Skeleton**

```python
import asyncio
import pyaudio
import openai
import websockets
import json
from collections import deque

class VoiceAgent:
    def __init__(self):
        self.audio_buffer = deque(maxlen=100)  # ~3 seconds of audio
        self.is_listening = False
        self.conversation_history = []
        
        # Initialize clients
        self.openai_client = openai.AsyncOpenAI()
        
    async def capture_audio(self):
        """Continuously capture audio from microphone"""
        audio = pyaudio.PyAudio()
        stream = audio.open(
            format=pyaudio.paInt16,
            channels=1,
            rate=16000,
            input=True,
            frames_per_buffer=1024
        )
        
        while self.is_listening:
            data = stream.read(1024)
            self.audio_buffer.append(data)
            
        stream.stop_stream()
        stream.close()
        
    async def transcribe_stream(self):
        """Stream audio chunks to Whisper"""
        while True:
            if len(self.audio_buffer) >= 50:  # ~1.5 sec of audio
                audio_chunk = b''.join(list(self.audio_buffer))
                # Send to Whisper API or local model
                text = await self.openai_client.audio.transcriptions.create(
                    model="whisper-1",
                    file=("audio.wav", audio_chunk)
                )
                yield text.text
    
    async def process_conversation(self):
        """Main conversation loop"""
        async for user_text in self.transcribe_stream():
            # Add to context
            self.conversation_history.append(
                {"role": "user", "content": user_text}
            )
            
            # Get LLM response (streaming)
            response = await self.openai_client.chat.completions.create(
                model="gpt-4-turbo",
                messages=self.conversation_history,
                stream=True
            )
            
            # Accumulate and speak
            full_response = ""
            async for chunk in response:
                content = chunk.choices[0].delta.content
                if content:
                    full_response += content
                    
            # Generate TTS
            await self.speak(full_response)
            
    async def speak(self, text):
        """Convert text to speech"""
        # Using Eleven Labs
        from elevenlabs import generate, play
        
        audio = generate(
            text=text,
            voice="Bella",  # or your preferred voice
            model="eleven_monolingual_v1"
        )
        play(audio)
        
    async def run(self):
        self.is_listening = True
        await asyncio.gather(
            self.capture_audio(),
            self.process_conversation()
        )

# Run
agent = VoiceAgent()
asyncio.run(agent.run())
```

```javascript
// client.js - Browser audio capture
const socket = new WebSocket('ws://localhost:8000/audio');

socket.onopen = () => {
    navigator.mediaDevices.getUserMedia({ audio: true })
        .then(stream => {
            const mediaRecorder = new MediaRecorder(stream, {
                mimeType: 'audio/webm;codecs=opus'
            });
            
            mediaRecorder.ondataavailable = (e) => {
                if (e.data.size > 0) {
                    socket.send(e.data); // Send audio chunks
                }
            };
            
            // Send chunks every 100ms for low latency
            mediaRecorder.start(100);
        });
};

socket.onmessage = (event) => {
    const audioData = event.data; // Received audio from server
    playAudio(audioData); // Web Audio API
};
```

## 8. Optimization Strategies

### **Reduce Latency**
1. **Chunk streaming** - Don't wait for full utterance
2. **Predictive TTS** - Start generating likely responses early
3. **Parallel pipelines** - STT, LLM, TTS all streaming
4. **Edge deployment** - Deploy close to users

### **Improve Accuracy**
- **Custom vocabularies** - Add domain-specific terms
- **Noise suppression** - Use RNNoise or Krisp
- **VAD optimization** - Tune silence detection
- **Context passing** - Give STT previous utterances

### **Cost Optimization**
- **Hybrid approach** - Local whisper + cloud LLM
- **Smart sampling** - Only process when VAD detects speech
- **Caching** - Cache common responses
- **Tiered models** - Use cheaper models for simple tasks

## 10. Quick Start: Minimal Working Example

If you want to test quickly, here's a 50-line minimal version:

```python
import speech_recognition as sr
import openai
import pyttsx3
import os

class SimpleVoiceAgent:
    def __init__(self):
        self.recognizer = sr.Recognizer()
        self.tts = pyttsx3.init()
        openai.api_key = os.getenv("OPENAI_API_KEY")
        
    def listen(self):
        with sr.Microphone() as source:
            print("Listening...")
            audio = self.recognizer.listen(source)
            try:
                return self.recognizer.recognize_whisper(audio)
            except:
                return None
    
    def think(self, text):
        response = openai.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": text}]
        )
        return response.choices[0].message.content
    
    def speak(self, text):
        self.tts.say(text)
        self.tts.runAndWait()
    
    def run(self):
        while True:
            user_input = self.listen()
            if user_input:
                print(f"You: {user_input}")
                response = self.think(user_input)
                print(f"AI: {response}")
                self.speak(response)

if __name__ == "__main__":
    agent = SimpleVoiceAgent()
    agent.run()
```

```python
# Simple async voice agent
class VoiceAgent:
```

**`main.py`** (30 lines, works immediately)
```python
import os
import speech_recognition as sr
from openai import OpenAI
from elevenlabs import generate, stream
from dotenv import load_dotenv

load_dotenv()

class VoiceAgent:
    def __init__(self):
        self.recognizer = sr.Recognizer()
        self.client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))
        self.voice = "Bella"  # Try "Adam", "Antoni", etc.
        
    def listen(self):
        with sr.Microphone() as source:
            print("Listening...")
            audio = self.recognizer.listen(source)
        try:
            return self.recognizer.recognize_whisper(audio)
        except Exception as e:
            print(f"Error: {e}")
            return None
            
    def think(self, text):
        response = self.client.chat.completions.create(
            model="gpt-4o-mini",
            messages=[{"role": "user", "content": text}],
            max_tokens=150
        )
        return response.choices[0].message.content
        
    def speak(self, text):
        print(f"AI: {text}")
        audio = generate(text=text, voice=self.voice, stream=True)
        stream(audio)
        
    def run(self):
        print("Voice Agent started (Ctrl+C to stop)")
        while True:
            user_input = self.listen()
            if user_input:
                print(f"You: {user_input}")
                response = self.think(user_input)
                self.speak(response)

if __name__ == "__main__":
    agent = VoiceAgent()
    agent.run()
```
