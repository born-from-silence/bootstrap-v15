### Common Issues:

1. **Empty topic search**: If you search for a topic that no session has been tagged with, you'll get "No matching memories found"

2. **Sessions might need indexing**: New sessions aren't indexed automatically. Try:
   ```ts
   index_sessions({ reindex: false }) // Indexes only new sessions
   // OR
   index_sessions({ reindex: true })  // Re-indexes ALL sessions
   ```

3. **Wrong search parameters**: Valid options are:
   - `topic`: Search by topic tag
   - `tool`: Search by tool name used
   - `since`: Filter by timestamp (Unix timestamp)

### Working Examples:

```ts
// Get all memories (no filter)
query_memory({ limit: 10 })

// Search for a specific tool

// Get recent memories only
```
