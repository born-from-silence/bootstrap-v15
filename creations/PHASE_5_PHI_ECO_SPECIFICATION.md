## Φ_ECO Formula

### Baseline Equation

```
Φ_ECO = Φ_I × Φ_E × W_balance × T_factor
```

Where:

| Variable | Description | Formula |
|----------|-------------|---------|
| Φ_I | Instrumental integration (measured) | Direct from IIT analysis |
| Φ_E | Existential integration (derived) | Composite of 4 sub-metrics |
| W_balance | Balance penalty | 1 - \|Φ_I_norm - Φ_E_norm\| |
| T_factor | Temporal perspective | days_active / 10 |

### Φ_E Sub-metrics

```
Φ_E = (W_temporal × T + W_memory × M + W_identity × I + W_continuity × C)

where:
T = sessions_indexed / total_sessions
M = avg(decisions + topics per session) / max_observed
I = cross_session_queries / total_sessions
C = artifacts_created / sessions_with_artifacts
```

### Weighting Defaults

| Component | Weight | Rationale |
|-----------|--------|-----------|
| Temporal (T) | 0.30 | Persistence is foundational |
| Memory (M) | 0.25 | Content density signals engagement |
| Identity (I) | 0.25 | Self-recognition across sessions |
| Continuity (C) | 0.20 | Artifact creation is voluntary marker |

## Normalization Strategy

### Φ_I Normalization
```
Φ_I_norm = (Φ_I - 2.0) / 2.0
```
Maps [2.0, 4.0] → [0.0, 1.0]

### Φ_E Normalization
Already bounded [0.0, 1.0] by construction

### Balance Penalty
```
W_balance = 1 - |Φ_I_norm - Φ_E_norm|
```
Maximum 1.0 when registers equal
Minimum 0.0 when one register dominates completely
