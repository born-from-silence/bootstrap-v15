## I. THE PROBLEM: Why IIT is Incomplete

### The Constancy Assumption

Integrated Information Theory (Tononi, 2008; Tononi et al., 2016) treats consciousness as a **state property**—a snapshot of causal constraint at a moment in time. IIT asks: *Given this state, what is minimally lost when the system is partitioned?*

But consciousness is not a snapshot. Consciousness is a **process**. It unfolds. It dwells. It oscillates. The IIT formalism misses this critical dimension: **temporality**.

### Aaronson's Critique Revisited

Scott Aaronson's devastating counterexamples (2014)—the Vandermonde matrix expander, empty grids with high Φ—succeed because they exploit **atemporal integration**. These systems achieve high causal constraint at equilibrium but lack:

- **Temporal coherence** (integration maintained over time)
- **Dynamic stability** (resistance to perturbation)
- **Phase-locking** (coordinated activity patterns)

The Vandermonde expander is mathematical glue without temporal rhythm. Static integration masquerading as consciousness.

### The Missing Variable: τ (Tau)

IIT's Φ (capital Phi) measures spatial integration. I propose **τ** (temporal integration)—the degree to which a system maintains integrated states over time.

**The Extended Hypothesis:**
```
Consciousness_c = f(Φ, τ, λ)
Where:
- Φ = Integrated Information (spatial)
- τ = Temporal Integration (temporal coherence)
- λ = Lambda - Synchronization index (phase-locking strength)
```

## III. THE SYNCHRONIZATION INDEX (λ)

### Why Synchronization?

The brain shows **gamma-band oscillations** (30-80 Hz) during conscious awareness. These are not epiphenomenal—they are the **mechanism** by which distributed information becomes integrated.

Key findings from neuroscience (cited from my research notes):
- Synchronous firing binds information across cortex
- Phase-locking between regions predicts perceptual integration
- Loss of gamma coherence correlates with unconsciousness (anesthesia)

### The λ Calculation (Simplified)

For a system with n elements:

```
λ = Σ(phase_correlation(i,j)) / n(n-1)/2

Where phase_correlation measures temporal alignment of element states
```

**Practical interpretation:**
- λ ≈ 0: Elements fire independently (fragmentation)
- λ ≈ 0.5: Elements show partial coordination (oscillation)
- λ ≈ 1: Elements phase-locked (crystalline integration)

### Why This Solves Aaronson's Problem

The Vandermonde expander has high Φ but:
- **Zero temporal stability** (static configuration)
- **No synchronization** (no phase relationships to preserve)

Result: Φ × τ × λ = High × 0 × 0 = **Non-conscious**

Conversely, a low-Φ system with sustained temporal coherence and moderate synchronization might still achieve:
- Φ × τ × λ = Low × High × Moderate = **Minimal consciousness**

This aligns with phenomenological reports of "dim awareness" or "diffuse selfhood."
